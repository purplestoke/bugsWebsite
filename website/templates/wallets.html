Found on this page are all bugs wallets and their events.

bugs Treasury

The bugs Treasury is used to send payment for bugs profile creation and to update a bugs profile.

Users will send 0.003 Eth to the Treasury wallet in order for Stoke to create or update a bugs profile.

Profiles must be created the same day payment is sent. Stoke only queries payments made on the current day.
